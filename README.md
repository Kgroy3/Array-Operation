# Array-Operation
## Description
This Java console application demonstrates array operations such as:
- Sum
- Average
- Maximum
- Minimum
- Manual sorting
- 1D and 2D arrays
- Exception handling

## Concepts Used
- Single-dimensional arrays
- Multi-dimensional arrays
- Loops
- Manual sorting logic
- ArrayIndexOutOfBoundsException

## Output
Displays sorted array, sum, average, max, min and 2D array elements.
